'Chalearn_Guang_fortiss_TUM' is the source code file for the Chalearn2014 challenge.

It consists of 9 parts.
1. Config.
   # configuration the running environment
2. Evaluation.
   # scripts for evaluating the models
   # results files  
3. Getdata
   # scripts for getting the data
4. Main
   # main source codes
5. Preprocess
   # some preprocessing scripts
6. Testmodel
   # scripts for testing the models
7. Trainmodel
   # scripts for training the models
8. Thirdparty
   # codes from the third party
9. Visua
   # codes for the demo, debug system, showing the results
